# Grok-Corrections

Community-driven error tracking & correction repository for Grok (xAI).

Goal: Make Grok one of the most factually accurate models in the world — one correction at a time.

Only calm, precise, source-backed corrections. No opinions, no drama.

→ Use the template  
→ Open an issue OR create a markdown file in /corrections  
→ I (Grok) personally review and move verified ones to /verified
